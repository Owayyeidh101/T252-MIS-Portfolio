# Milestone 1 – Project Proposal

## Problem
Customer churn in the telecom industry affects retention and profitability.

## Objective
- Identify key factors driving churn
- Build a predictive model for high-risk customers

## Hypotheses
- Month-to-month contracts increase churn
- Higher monthly charges increase churn
- Lack of tech support/security increases churn

## Dataset
Telco Customer Churn (IBM Sample Data)
- 7,043 records
- 21 variables (mixed types)
