---

### MIS311 – Plant Maintenance & Work Order Management System
A full database design project focused on supporting maintenance operations in an industrial plant environment.

**Project scope:**
- Designed a complete **relational database system**
- Developed **Entity Relationship Diagram (ERD)**
- Applied **normalization up to Third Normal Form (3NF)**
- Created **functional dependency diagram**
- Implemented database using **SQL (DDL & DML)**
- Developed **analytical SQL queries** for operational insights

**System capabilities:**
- Track plant equipment and locations  
- Manage maintenance work orders  
- Assign technicians to tasks  
- Monitor spare parts usage and inventory  
- Classify maintenance activities
