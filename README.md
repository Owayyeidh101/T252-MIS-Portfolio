# T252 MIS Portfolio (KFUPM)

This repository presents selected technical coursework completed during Term 252 as part of the Management Information Systems (MIS) program at King Fahd University of Petroleum and Minerals (KFUPM). The work focuses on practical applications in data science and database design.

---

## Focus Areas

* Data Science and Analytics (ISE291)
* Database Design, Data Modeling, and SQL (MIS311)

---

## Highlight Projects

### ISE291 – Telco Customer Churn Analysis

A comprehensive data science project analyzing customer churn behavior in the telecommunications industry. The project applies the full data analytics lifecycle to generate insights and build predictive models.

**Project workflow:**

* Problem definition and hypothesis formulation
* Data collection and cleaning
* Exploratory Data Analysis (EDA)
* Model development, evaluation, and interpretation

**Key outcomes:**

* Identification of major churn drivers such as contract type, tenure, and monthly charges
* Development of classification models to predict churn risk
* Data-driven recommendations to support customer retention strategies

---

### MIS311 – Plant Maintenance & Work Order Management System

A full database design project focused on supporting maintenance operations in an industrial plant environment.

**Project scope:**

* Designed a complete **relational database system**
* Developed **Entity Relationship Diagram (ERD)**
* Applied **normalization up to Third Normal Form (3NF)**
* Created **functional dependency diagram**
* Implemented database using **SQL (DDL & DML)**
* Developed **analytical SQL queries** for operational insights

**System capabilities:**

* Track plant equipment and locations
* Manage maintenance work orders
* Assign technicians to tasks
* Monitor spare parts usage and inventory
* Classify maintenance activities
